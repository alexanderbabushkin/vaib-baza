# Документы: Вводный блок «Про ИИ и аналитику»/3. Вводный об агентах

## Источник: [SuperSliv.biz] Документ Microsoft Word .docx

Для каких целей создаем агентов? 
• Контент, аналитика, клиентский сервис, помощь в запуске
• Агент для Reels, агент для аналитики, агент для консультаций
• Сценарий работы: идея → агент → результат
Можно вообще для разных целей!
Где применяем агентов в бизнесе: 

• Придумывает идеи, пишет тексты, анализирует
• Помощник в рабочих задачах
• Используем для сокращения времени и упрощения рутины
• Роль: как «второй мозг»
Генерация идей: 

Агент придумывает идеи постов, сторис, вебинаров, прогревов. Reels без затыков и креативного выгорания — он всегда готов генерить свежак.

Пример: «GPT, дай 10 идей для сторис по теме "страхи продаж" с вовлечением и эмоциями» (но такой запрос без промта даем уже обученному агенту).

Написание текстов: 
• Пишет посты, письма, прогревы, сторис, тексты в боты, даже скрипты продаж
• Сокращает время в 3-5 раз
Пример: «Создай текст сторис по модели "боль — интрига — решение" для аудитории новичков в бизнесе».
Анализ данных:
• Анализ анкет, опросов, воронок, цифр
• Выводит слабые места и дает идеи, что докрутить
Пример: «Проанализируй воронку: [описание]. Где теряются клиенты и как это улучшить?»

Помощник в рабочих задачах:
• Планирование недели, чек-листы, пошаговые инструкции, контроль задач
• Структурирует хаос и помогает не выгорать
Пример: «Разбей эти задачи на неделю, добавь зоны отдыха и напомни, что важно сделать в первую очередь».
